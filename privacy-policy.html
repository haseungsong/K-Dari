K-Dari – Privacy Policy

Last Updated: 2025-11-01

1. Information We Collect

K-Dari (“the App”) collects the following information to provide and improve the service:

1) Account Information

  • Email address
  • Firebase Authentication account information
  • Passwords are encrypted and processed only by Firebase Authentication. The app does not store password data.

2) App Usage Data

  • Usage logs
  • Selected learning items
  • Error and crash logs (Firebase Crashlytics)

3) Device Information

  • Device model and OS version
  • App version
  • Advertising ID (for AdMob ads)

4) Audio Data

  • Microphone input used for pronunciation test features
  • Audio is not stored and is discarded after processing

2. How We Collect Information

  • Information provided during login
  • Automatically collected data through Firebase
  • Microphone input during pronunciation testing

3. How We Use Information

  • User authentication
  • Learning feature operation
  • Advertisement delivery
  • App performance improvement
  • Pronunciation analysis

4. Sharing Information with Third Parties

■ Google Firebase

  • Purpose: authentication, analytics, crash reporting
  • Location: USA
  • Privacy Policy: https://firebase.google.com/support/privacy

■ Google AdMob

  • Purpose: banner advertisement
  • Shared data: Advertising ID, device data
  • Privacy Policy: https://policies.google.com/technologies/ads

No other third party receives your data.

5. Data Retention & Deletion

  • Account data is deleted immediately upon user request.
  • Analytics and crash logs follow Firebase retention policies.
  • Audio data is not stored.

6. User Rights

  • Request account deletion
  • Request access or correction of personal data
  • Request withdrawal of consent

Contact: support@k-dari.app

7. Data Protection Measures

  • Encrypted data transmission
  • Firebase Security Rules applied
  • No sensitive data stored externally

8. Children’s Privacy

The App is intended for users aged 18 and older. We do not knowingly collect data from children.

9. Changes to This Policy

We may revise this policy. Changes will be posted with an updated date.

10. Contact Us

Email: haseungbr@gmail.com

K-Dari – 개인정보처리방침

최종 업데이트: 2025-11-01

1. 수집하는 개인정보

K-Dari(“본 앱”)는 서비스 제공을 위해 다음 정보를 수집할 수 있습니다.

1) 계정 정보

  • 이메일 주소
  • Firebase 인증 정보
  • 비밀번호는 Firebase에서 암호화되어 관리되며, 앱은 저장하지 않습니다.

2) 앱 사용 데이터

  • 앱 사용 기록
  • 선택한 학습 항목
  • 오류 및 충돌 로그

3) 기기 정보

  • 기기 모델, OS 버전
  • 앱 버전
  • 광고 ID (AdMob 배너 광고 제공 목적)

4) 오디오 데이터

  • 발음 테스트 기능 사용 시 마이크 음성
  • 오디오는 저장되지 않으며 처리 후 즉시 폐기됩니다.

2. 수집 방법

  • 로그인 시 직접 제공
  • Firebase를 통한 자동 수집
  • 발음 테스트 중 마이크 사용

3. 개인정보 이용 목적

  • 사용자 인증
  • 학습 기능 제공
  • 광고 제공
  • 앱 성능 개선
  • 발음 분석

4. 개인정보 제3자 제공

■ Google Firebase

  • 용도: 인증, 통계, 오류 분석
  • 위치: 미국
  • 정책: https://firebase.google.com/support/privacy

■ Google AdMob

  • 용도: 배너 광고 제공
  • 공유 정보: 광고 ID, 기기 정보
  • 정책: https://policies.google.com/technologies/ads

그 외 제3자에게 정보를 제공하지 않습니다.

5. 보관 및 삭제

  • 사용자 요청 시 계정 즉시 삭제
